Discuss the scope of var, let, and const

var:

var is a functional-level scope. They are only access within the function

let:

let is a block-level scope. They are only access within the block where they are declared.

const:

const also a block-level scope. once const variable is declared and this value can not reassigned.

Tell us the use cases of null and undefined

null

null is a primitive data type, often used to indicate that a variable, property, or value should have no values or should be empty

undefined

undefined is a primitive value that indicates the absence of a value for a given variable or property.
